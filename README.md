# music_store_analysis

## Project Overview
This project performs SQL-based analysis on a music store database to answer various business-related analytical questions related to customers, sales, invoices, artists, genres, and employee performance.

The project was created using MySQL and focuses on extracting meaningful insights from structured relational data using SQL queries.

## Tools Used
- MySQL
- SQL

## SQL Concepts Used
- SELECT Statements
- WHERE Clause
- GROUP BY
- ORDER BY
- Aggregate Functions
- JOINS
- Common Table Expressions (CTEs)
- Window Functions
- Subqueries

## Business Questions Solved

### 1. Senior Most Employee
Identified the senior-most employee based on job title hierarchy.

### 2. Countries with Most Invoices
Analyzed which countries generated the highest number of invoices.

### 3. Top Invoice Totals
Retrieved the top invoice values from the dataset.

### 4. City with Highest Revenue
Found the city generating the highest total invoice revenue.

### 5. Best Customer
Identified the customer who spent the most money overall.

### 6. Rock Music Listeners
Retrieved customer details of users who listen to Rock music.

### 7. Top Rock Artists
Found the top 10 artists with the highest number of Rock tracks.

### 8. Tracks Longer Than Average
Listed songs with duration longer than average track length.

### 9. Customer Spending on Artists
Calculated amount spent by customers on music artists.

### 10. Most Popular Genre by Country
Determined the most purchased music genre for each country.

### 11. Top Customer by Country
Found the highest spending customer from each country.

## Key Insights
- Certain countries generated significantly higher invoice counts and revenue.
- Rock music was one of the most dominant genres in the dataset.
- Some customers contributed disproportionately high revenue.
- Sales trends varied across cities, countries, and music genres.
